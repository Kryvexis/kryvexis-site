FAQ

Quick answers to common questions.

Is this built for South African businesses?
Yes. Kryvexis is designed for the way small businesses actually work day-to-day — simple, fast, and practical.
Can we start small and upgrade later?
Absolutely. Start on Starter, then unlock Premium/Enterprise features when you need them.
Do you support printing and PDF invoices?
Yes — one-click printing and automatic PDF saving are supported per tier / roadmap.
How do we get onboarded?
We help you load products, map suppliers, and set up dashboards in a quick onboarding session.

Yes. We can add an import flow so you don’t capture sales twice.
